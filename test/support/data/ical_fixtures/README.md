# iCal fixtures

These `.ics` files are copied from the test suite of the [`ical`](https://github.com/expothecary/ical) Elixir library (MIT licence, copyright Aaron Seigo, Max Salminen, and contributors; originally Louis Pilfold). They provide small, public, RFC-5545-valid examples for exercising `Tempo.ICal.from_ical/2`.

Attribution is preserved here rather than in each individual file. If you add more fixtures from other sources, note them here with their licence.
